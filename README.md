# Rational-Geometry-in-Modular-3D
A complete geometric engine for builders and devs. The Symmetric Span Protocol (SSP) restores the missing origin to the Pythagorean theorem, replacing complex trig with a simple symmetry matrix. Build stable 3D modules using pure arithmetic. From the 12-knot rope to modular code, it’s a coordinate-first toolkit for rational construction.
